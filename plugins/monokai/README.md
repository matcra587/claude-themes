# Monokai

The classic Sublime Text Monokai palette ported to Claude.

## Installation

```text
/plugin install monokai@matcra587/claude-themes
```

## Themes

| Theme          | Base        | Source                       |
| -------------- | ----------- | ---------------------------- |
| Monokai        | `dark`      | Classic Sublime Text Monokai |
| Monokai (ANSI) | `dark-ansi` | Classic Sublime Text Monokai |

## Variants

Ships in regular and **(ANSI)** forms. The ANSI variant delegates code-block syntax highlighting to your terminal's ANSI palette. See the [marketplace README](https://github.com/matcra587/claude-themes#ansi-variants) for details.
