# Solarized

Ethan Schoonover's canonical Solarized palette — both Dark and Light modes — ported to Claude.

## Installation

```text
/plugin install solarized@matcra587/claude-themes
```

## Themes

| Theme                  | Base         | Source                                         |
| ---------------------- | ------------ | ---------------------------------------------- |
| Solarized Dark         | `dark`       | Ethan Schoonover's canonical Solarized palette |
| Solarized Dark (ANSI)  | `dark-ansi`  | Ethan Schoonover's canonical Solarized palette |
| Solarized Light        | `light`      | Ethan Schoonover's canonical Solarized palette |
| Solarized Light (ANSI) | `light-ansi` | Ethan Schoonover's canonical Solarized palette |

## Variants

Each mode ships in regular and **(ANSI)** forms. The ANSI variant delegates code-block syntax highlighting to your terminal's ANSI palette. See the [marketplace README](https://github.com/matcra587/claude-themes#ansi-variants) for details.
