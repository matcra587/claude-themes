# Tokyo Night

The Tokyo Night palette (the `night` style from `tokyonight.nvim`) ported to Claude.

## Installation

```text
/plugin install tokyo-night@matcra587/claude-themes
```

## Themes

| Theme              | Base        | Source                          |
| ------------------ | ----------- | ------------------------------- |
| Tokyo Night        | `dark`      | `tokyonight.nvim` `night` style |
| Tokyo Night (ANSI) | `dark-ansi` | `tokyonight.nvim` `night` style |

## Variants

Ships in regular and **(ANSI)** forms. The ANSI variant delegates code-block syntax highlighting to your terminal's ANSI palette. See the [marketplace README](https://github.com/matcra587/claude-themes#ansi-variants) for details.
