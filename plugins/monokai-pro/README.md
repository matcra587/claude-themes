# Monokai Pro

The Monokai Pro palette (the `pro` filter from `monokai-pro.nvim`) ported to Claude.

## Installation

```text
/plugin install monokai-pro@matcra587/claude-themes
```

## Themes

| Theme              | Base        | Source                          |
| ------------------ | ----------- | ------------------------------- |
| Monokai Pro        | `dark`      | `monokai-pro.nvim` `pro` filter |
| Monokai Pro (ANSI) | `dark-ansi` | `monokai-pro.nvim` `pro` filter |

## Variants

Ships in regular and **(ANSI)** forms. The ANSI variant delegates code-block syntax highlighting to your terminal's ANSI palette. See the [marketplace README](https://github.com/matcra587/claude-themes#ansi-variants) for details.
